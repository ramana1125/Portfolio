# 🚀 Personal Portfolio Website

A modern and responsive full-stack portfolio website built using the MERN stack.  
This portfolio showcases my projects, technical skills, certifications, education, and experience with a clean UI and interactive animations.

---

## 📌 Features

- Responsive modern UI
- Animated hero section
- Interactive custom cursor
- Projects showcase section
- Skills & tech stack section
- Certifications section
- Education & experience timeline
- Contact/Connect section
- Resume download option
- Full-stack MERN architecture

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- CSS3

### Backend
- Node.js
- Express.js

### Tools & Platforms
- Git & GitHub
- VS Code
